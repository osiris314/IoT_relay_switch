# IoT_relay_switch
Control switches over wifi with an arduino

parts:

0. multimeter
1. arduino
2. esp8266(wifi_chip)
3. some small cables
4. some high voltage cables
5. wall socket cable
6. wall sockets
7. relay board
8. usb cable
9. phone power adapter
10. box to install them in
